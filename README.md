

# UART-Based DC Motor Control Using QNX RTOS

## Overview
This project demonstrates a UART-based DC motor control system implemented in a QNX RTOS virtual environment.

The system simulates real-time motor control command handling using UART communication principles and embedded C programming.

## Features
- UART command handling
- Real-time task execution
- Motor ON/OFF simulation
- Speed control logic
- RTOS-based software workflow

## Technologies Used
- QNX Neutrino RTOS
- Embedded C
- UART Communication
- Virtual Machine Environment

## Sample Commands
- 1 → Motor ON
- 0 → Motor OFF
- 2 → Increase Speed

## Output
The terminal displays UART command reception and motor state transitions in real time.

## Future Scope
The project can be extended to physical hardware deployment using:
- Raspberry Pi
- ESP32
- L298N Motor Driver
- DC Motor
